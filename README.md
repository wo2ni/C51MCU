## 如何使用:
硬件设备,***NewWifi3无线路由器***一台;***普通无线路由器***一台;***RJ45_T568B千兆网线***一根;
(***声明:为了追求极致&稳定,内核默认去掉了无线芯片的驱动,以及USB3.0的驱动;***)
```ruby
1,重新刷入"BootLoader"请刷入主流的"Breed"这很简单,不介绍,
2,更改掉所有的MAC地址,记得备份好EEPROM;
3,刷入此固件;
4,Reboot and Enjoy;
```
